## 프로젝트 개요

OOTD AI는 사용자가 오늘의 착장(OOTD) 사진을 업로드하면 AI가 패션 점수, 강점, 개선점, 스타일링 팁을 즉시 분석해주고, 추출한 옷을 스마트 옷장에 자동 등록하며, 옷장 데이터를 기반으로 날씨에 맞는 AI 코디를 추천해주는 올인원 패션 AI 웹 서비스입니다.

---

## 주요 기능

### 1. OOTD AI 분석
- 사용자가 착장 사진을 업로드하면 Gemini 멀티모달 AI가 심층 분석
- 전체 점수(0–100) 및 핏·실루엣 / 컬러 조합 / 스타일링 완성도 / 날씨 적합도 4개 카테고리 세부 채점
- 헤드라인 한 줄 평, 잘된 점 2가지, 개선점 2가지, 실행 가능한 스타일링 팁 2가지, 날씨 적합성 코멘트 제공
- 분석 결과를 OOTD 피드로 저장하여 나중에 다시 열람 가능

### 2. AI 스트리밍 UX
- Gemini 응답을 스트리밍으로 수신하여 결과 패널이 즉시 열림
- 데이터가 도착하는 순서대로 스켈레톤 → 실제 값으로 자연스럽게 전환
- 총 분석 소요 시간을 체감적으로 크게 단축

### 3. AI 옷장 자동 등록 (스마트 스캐너)
- OOTD 분석 결과에서 "옷장에 자동 등록하기" 버튼 한 번으로 옷 추출 시작
- **Phase 1:** Gemini Vision으로 사진 내 의류 아이템별 위치(Y좌표) 감지
- **Phase 2:** `@imgly/background-removal` 라이브러리로 클라이언트 사이드 배경 제거 (서버 비용 없음)
- **Phase 3:** Gemini가 각 아이템의 색상·스타일·종류를 조합한 한국어 이름 자동 생성
- **수동 영역 선택:** AI가 놓치거나 불필요한 부분을 드래그로 직접 영역 지정해 재추출 가능

### 4. AI 코디 큐레이션
- 사용자 옷장 데이터, 현재 날씨, 개인 체형/스타일 정보를 Gemini에 전달
- 날씨·핏·무드에 최적화된 코디 조합 자동 추천
- 추천 아이템별 무신사, 29CM 쇼핑 링크 연동
- 코디 피드백(좋아요/싫어요) 저장으로 점진적 개인화

### 5. 날씨 연동
- Geolocation API로 현재 위치를 감지하여 실시간 날씨 정보를 분석·큐레이션에 반영
- 날씨 기반 스타일 팁 토스트 알림 제공

### 6. 사용자 프로필 및 개인화
- 온보딩에서 키·몸무게·핏 선호·스타일 무드·바디 목표를 입력하면 AI 분석·큐레이션에 반영
- Supabase Storage에 프로필 이미지 업로드 및 관리

## 기술적 특징

### Gemini 멀티모달 스트리밍
```
Gemini API의 generateContentStream()을 사용하여 응답을 청크 단위로 수신.
서버에서 ReadableStream으로 클라이언트에 전달하고, 클라이언트는 정규식 기반
부분 JSON 파싱으로 도착한 필드부터 즉시 UI에 렌더링.
```

### 클라이언트 사이드 배경 제거
```
@imgly/background-removal은 WebAssembly 모델을 브라우저에서 직접 실행.
서버 API 호출 없이 클라이언트에서 배경 제거를 처리하므로 서버 비용 없음.
4단계 전략(해상도·패딩·대비 조합)으로 실패 시 자동 재시도, 최종 실패 시 크롭 이미지로 fallback.
```

### Supabase SSR 세션 관리
```
Next.js 16의 proxy.ts(middleware 역할)에서 @supabase/ssr의 updateSession으로
모든 요청마다 세션 자동 갱신. 브라우저를 닫았다 열어도 로그인 상태 유지.
```

### 일일 사용량 제한 (Rate Limiting)
```
Supabase DB에 user_id + action + date 조합으로 사용 횟수를 기록.
OOTD 분석·옷 추출·이름 생성 각각 일일 한도를 적용하여 API 비용 제어.
```

### 이미지 최적화
```
FileReader로 읽은 원본 이미지를 Canvas API로 768px 이하 WebP로 압축 후
Gemini API에 전송. sessionStorage 한도(5MB) 초과 방지를 위해 옷장 이동 시에도
1024px JPEG로 재압축.
```

### 모바일 퍼스트 / 반응형 UI
```
모바일: 사진 오버레이에 총점 배지 + 하단 슬라이딩 시트 구조
데스크탑: 좌측 사진 + 우측 분석 결과 사이드바 구조
Tailwind CSS의 lg: 분기점으로 두 레이아웃을 단일 코드베이스에서 관리.
```

## 서비스 흐름

```
로그인 → 온보딩(체형/스타일 입력)
    ↓
[OOTD 분석 탭]
사진 업로드 → Gemini 스트리밍 분석 → 결과 패널 (점수·헤드라인·피드백)
    → 저장 → OOTD 피드
    → 옷장에 자동 등록하기 → 세그멘테이션 → 배경 제거 → AI 이름 생성 → 옷장 저장

[AI 코디 탭]
옷장 데이터 + 날씨 + 프로필 → Gemini 코디 생성 → 코디 카드 (아이템별 쇼핑 링크)
    → 피드백 저장
```
